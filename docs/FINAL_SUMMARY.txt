WHAT WAS BUILT
================================================================================

A production-ready Ruby CLI application that:

1. CONNECTS to RTSP streams from IP cameras
2. CAPTURES every RTP packet in real-time
3. ANALYZES each packet for:
   - Wallclock timestamp (milliseconds)
   - Packet number
   - Packet type (H264, H265, JPEG, etc.)
   - Raw packet size
   - Stream timestamp
   - Timestamp deviation (jitter detection)
4. LOGS data in multiple formats (JSON, CSV, Binary)
5. SAVES raw stream for forensic analysis

================================================================================
LOGGED PACKET INFORMATION
================================================================================

Each packet is analyzed and logged with:

1. packet_number - Sequential packet ID
2. wallclock_time_ms - System timestamp (milliseconds)
3. packet_type - RTP payload type name (H264, H265, etc.)
4. payload_type_code - Numeric payload type code
5. raw_packet_size - Total packet size in bytes
6. stream_timestamp - RTP timestamp from packet
7. sequence_number - RTP sequence number
8. marker_bit - RTP marker bit (0 or 1)
9. ssrc - Synchronization source identifier
10. timestamp_deviation_ms - Jitter/deviation metric

================================================================================
QUICK START
================================================================================

1. Install Dependencies:
   $ cd streamripper-rb
   $ gem install bundler
   $ bundle install
   $ chmod +x bin/streamripper

2. Verify Installation:
   $ bundle exec rspec

3. Capture Stream:
   $ ./bin/streamripper capture rtsp://192.168.1.100:554/stream

4. View Results:
   $ cat logs/stream_analysis.json | jq '.'

================================================================================
COMMAND-LINE OPTIONS
================================================================================

./bin/streamripper capture <URL> [OPTIONS]

Options:
  -o, --output FILE              Output file for analysis
  -r, --raw-stream FILE          Output file for raw stream
  -f, --format FORMAT            Output format (json/csv/both)
  -d, --duration SECONDS         Capture duration (0 = infinite)
  -m, --max-packets COUNT        Maximum packets (0 = infinite)
  -v, --verbose                  Enable verbose logging

Examples:
  # Basic capture
  ./bin/streamripper capture rtsp://192.168.1.100:554/stream

  # With duration
  ./bin/streamripper capture rtsp://192.168.1.100:554/stream --duration 60

  # CSV format
  ./bin/streamripper capture rtsp://192.168.1.100:554/stream --format csv

  # With authentication
  ./bin/streamripper capture rtsp://admin:password@192.168.1.100:554/stream
